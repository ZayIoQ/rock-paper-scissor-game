# Python Mini Games

Коллекция консольных игр на Python. В этом проекте я тренирую чистоту кода и использование продвинутых структур данных вместо бесконечных условий.

## Игры в наборе:
1. **Rock Paper Scissors (Камень, Ножницы, Бумага) 2.0**
   - Реализована с использованием словаря (Dictionary Mapping) для проверки условий победы.
   - Поддерживает бесконечный цикл игры с возможностью корректного выхода.
   - Очистка ввода: программа понимает текст в любом регистре и обрезает лишние пробелы.

## Технологии и фишки:
* **Python 3.13**
* **Dictionary Mapping:** Вместо 9-ти блоков `if-elif-else`, вся логика игры умещается в одну строку.
* **Error Handling:** Проверка на корректность ввода пользователя.
* **Infinite Loops:** Управление игровым процессом через `while True` и `break`.

## Как запустить:
1. Убедитесь, что у вас установлен Python.
2. Скачайте файл `rps_game.py`.
3. Запустите его через терминал:
   ```bash
   python rps.py
